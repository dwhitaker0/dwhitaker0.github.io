####functions####
avgspectra = function(df, n = 3){ #average each n rows
  aggregate(x = df,
            by = list(gl(ceiling(nrow(df)/n), n)[1:nrow(df)]),
            FUN = mean)
}

Each probe takes 5 scans, conventionally these are combined and averaged to improve signal to noise ratio the multieye software doesn't do this yet, the function defined eariler takes care of this

The package 'hyperSpec' provides a good set of commands for manipulating spectral data (NIR, Raman, UV, Mass, NMR . .) The data is added on a per probe basis to its own hyperSpec object.

####Create hyperSpec Objects####

P1spc <- new("hyperSpec", spc=Probe1Avg[,-1], wavelength = Wavelength,
    labels = list (.wavelength = "nm", spc = "A"))
